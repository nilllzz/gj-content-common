# Mark: Code

| Field                | Value  |
| -------------------- | ------ |
| **Mark Object Type** | `code` |

---

Displays decorated text as inline code with a fixed-width font.

## Attributes

(none)

## Example

```
{
    "type": "text",
    "text": "Hello, World!",
    "marks: [
        {
            "type": "code",
        }
    ],
}
```

#### Produces:

`Hello, World!`
