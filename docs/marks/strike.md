# Mark: Strike

| Field                | Value    |
| -------------------- | -------- |
| **Mark Object Type** | `strike` |

---

Displays decorated text as strikethrough.

## Attributes

(none)

## Example

```
{
    "type": "text",
    "text": "Hello, World!",
    "marks: [
        {
            "type": "strike",
        }
    ],
}
```

#### Produces:

~~Hello, World!~~
