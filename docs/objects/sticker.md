# Object: Sticker

| Field                   | Value     |
| ----------------------- | --------- |
| **Content Object Type** | `sticker` |
| **Group**               | `block`   |
| **Valid children**      | (none)    |

---

Displays a Game Jolt Sticker.

## Attributes

| Name | Description                         |
| ---- | ----------------------------------- |
| `id` | Numeric id of the Game Jolt Sticker |

## Example

```
{
    "type": "sticker",
    "attrs": {
        "id": 1234
    }
}
```
